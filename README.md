# iOS-URL-schemes
A README file that contains a list of URL scheme for iOS


## iOS-URL-schemes list

### Browsers
  Chrome - ```googlechrome://```
    - Note, when using this scheme do not include the http/https prefix, because the google chrome scheme already provides one.
